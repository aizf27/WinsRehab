# WinsRehab - 智慧康复APP

WinsRehab 是一个基于Android的康复训练管理APP，为医生和患者提供便捷的康复训练视频管理、患者信息跟踪和AI心理咨询服务。

## 🌟 功能特性

### 医生端功能
- **患者管理**：查看和管理负责的患者列表
- **患者信息**：了解患者基本信息并维护康复进度
- **康复视频管理**：为患者分配和跟踪康复训练视频（待开发）
- **数据统计**：查看患者康复数据和统计信息

### 患者端功能
- **个人信息管理**：维护个人基本信息并关注康复数据
- **康复视频训练**：观看医生分配的康复训练视频
- **AI心理咨询**：与AI心理咨询师进行对话交流
- **康复进度跟踪**：查看个人康复进度和分析结果

### 系统特色
- **双端设计**：医生端和患者端分离，功能明确
- **本地数据库**：使用Room数据库进行数据持久化
- **视频播放**：支持本地assets
- **AI集成**：集成AI心理咨询服务

## 🏗️ 技术架构

### 核心技术栈
- **开发语言**：Kotlin
- **架构模式**：MVVM (Model-View-ViewModel)
- **数据库**：Room Database
- **异步处理**：Kotlin  Flow
- **网络请求**：OkHttp和Gson
- **UI框架**：XML

### 主要组件
- `Room Database`：本地数据存储
- `ViewModel`：UI数据管理和生命周期感知
- `LiveData/Flow`：数据流处理
- `RecyclerView`：列表展示
- `VideoView`：视频播放支持

## 🚀 快速开始

### 环境要求
- Android Studio Arctic Fox 或更高版本
- Kotlin 1.5+
- Android SDK API 21+

### 安装步骤
1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 同步Gradle依赖
4. 构建并运行应用

### 使用说明
1. 首次使用需要注册账号（医生/患者）
2. 登录后根据角色进入相应功能界面
3. 医生可以管理患者
4. 患者可以查看训练视频和进行心理咨询

## 📱 主要界面

- **登录界面**：医生和患者分别登录
- **医生主页**：患者列表和信息管理
- **患者主页**：康复训练和心理咨询服务
- **视频播放器**：支持本地和网络视频播放
- **个人信息**：用户信息维护界面
